RAR-Archive-Utility by BlackWolf

A fast, minimalistic (but powerful), easy-to-use RAR-Extractor and -Archiver I created because there seem to be no good RAR-Utilities out there for Mac OS X

After coming up empty in my search for good RAR-Utilites for Mac OS X, I decided to simply write my own. I looked at what Apple did with their built-in ZIP-Utility and tried to get the same feel for my RAR-Tool. Which means: Common tasks like extracting or creating RAR-Files should take as less user interaction as possible, while the UI keeps a minimalistic style (and - hopefully - follows Apple's UI-Guidelines). On the other hand, I wanted the Tool to be powerful if the user wants it to be. So after working a couple of months in my free time, this is what I came up with. I wrote this for personal usage and I'm NOT an Objective-C-Pro, so the code might be a little messy (though I tried to keep it clean). Also, the idea to publish it came after I'd begun working on the project. Still, I think the Tool is really great which is why I wanted to share it with people who might be interested in this. Dependencies

Mac OS X 10.6.0 or newer

Install

To just use it: Go into the build/Release/ folder and grab whichever version you'd like to use Compiling it: Open the .xcodeproj file with XCode, then hit cmd+r to compile
